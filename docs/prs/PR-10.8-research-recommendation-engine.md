# PR-10.8 — Research Recommendation Engine

## CLI

```bash
npm run research:recommendations
```

Optional flags: `--portfolio-analytics`, `--roi-analysis`, `--interaction-analysis`, `--dimension-explorer`, `--failure-analysis`, `--month-regime-analysis`, `--output`, `--html-output`.

## Inputs (all optional)

| Artifact | Default path |
|----------|--------------|
| Portfolio analytics | `data/research-results/research-portfolio-analytics.json` |
| ROI analysis | `data/research-results/research-roi-analysis.json` |
| Interaction analysis | `data/research-results/research-interaction-analysis.json` |
| Dimension explorer | `data/research-results/research-dimension-explorer.json` |
| Failure analysis | `data/research-results/hypothesis-failure-analysis.json` |
| Month regime analysis | `data/research-results/month-regime-analysis.json` |

Missing artifacts do not fail execution.

## Outputs

- `data/research-results/research-recommendations.json`
- `data/reports/research-recommendations.html`

## Recommendation kinds

Heuristic, deterministic recommendations with explanations:

- Expand research family (e.g. Momentum)
- Reduce exploration focus (e.g. Hour-only)
- Split dimension buckets (e.g. Probability)
- Increase sampling window (e.g. Weekend)
- Investigate interaction (e.g. Momentum × Volatility)
- Deprioritize sparse dimensions
- Recommend registry dimensions
- Recommend interaction families
- Recommend refinement priorities

## Constraints

Read-only diagnostics. Does not modify validation, hypothesis generation, or strategy synthesis.
