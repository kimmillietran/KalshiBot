# PR 11.4 — Feature Catalog Explorer

## Summary

Read-only CLI report that joins the unified feature catalog (M11.1) with the research dimension registry and optional diagnostics.

## CLI

```bash
npm run research:feature-catalog
```

Optional flags:

- `--output` — JSON path (default `data/research-results/feature-catalog.json`)
- `--html-output` — HTML path (default `data/reports/feature-catalog.html`)
- `--dimension-explorer` — optional dimension explorer JSON
- `--portfolio-analytics` — optional portfolio analytics JSON
- `--roi-analysis` — optional ROI analysis JSON
- `--duplication-analysis` — optional duplication analysis JSON

## Outputs

- `data/research-results/feature-catalog.json`
- `data/reports/feature-catalog.html`

## Architecture

- `src/lib/data/research/featureCatalog/` — unified catalog (M11.1)
- `src/lib/data/research/featureCatalogExplorer/` — report builder, optional input loader, serializers
- `scripts/research/buildFeatureCatalogExplorer.ts` — CLI entrypoint
