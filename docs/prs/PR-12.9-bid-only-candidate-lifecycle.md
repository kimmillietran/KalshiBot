# PR-12.9 — Bid-Only Candidate Lifecycle Builder

## Summary

Groups M12.7 bid-only parity scan records into offline candidate episodes with duration, edge, size, BTC, and time-to-close diagnostics.

## CLI

```bash
npm run research:bid-only-candidate-lifecycle
```

Outputs:

- `data/research-results/bid-only-candidate-lifecycle.json`
- `data/reports/bid-only-candidate-lifecycle.html`

## Episode construction

Episodes group consecutive records by `marketTicker`, classification family, and gap threshold (default 2,500ms).
