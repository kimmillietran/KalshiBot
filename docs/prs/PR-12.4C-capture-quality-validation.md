# PR-12.4C — Capture Quality Validation Harness

## Summary

Adds an independent offline validation harness that scans `data/live-capture/forward-quotes/` and compares capture health JSON counts against recomputed economic validity from `top-of-book.jsonl`.

Reuses M12.4B `classifyTopOfBookEconomicValidity` for validation recomputation. Supports legacy captures (no `economicBookState`) and new M12.4B economic-state fields with mismatch detection.

## CLI

```bash
npm run research:capture-quality-validation
# or
npx tsx scripts/research/buildCaptureQualityValidation.ts
```

Outputs:

- `data/research-results/capture-quality-validation.json`
- `data/reports/capture-quality-validation.html`
