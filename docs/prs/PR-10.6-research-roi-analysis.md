# PR-10.6 — Research ROI Analysis

## Summary

Adds a **read-only** research ROI report that measures whether hypothesis research effort is paying off, using existing artifacts only. No changes to hypothesis generation, validation, imports, or replay.

## Metrics

**Overall**
- Candidate generation efficiency (candidates / atlas buckets)
- Validated / candidates, near-promising / candidates
- Average robustness, validation efficiency, bucket utilization
- Average robustness improvement after refinement (parent vs validated child)

**Per dimension / axis group / bucket**
- Candidate yield, validation rate, near-promising rate
- Bucket utilization, ROI score, research cost score, efficiency score

## Rankings

- Highest / lowest ROI dimensions
- Highest ROI axis groups
- Most expensive research areas
- Most efficient research areas

## Outputs

- `data/research-results/research-roi-analysis.json`
- `data/reports/research-roi-analysis.html`

## CLI

```bash
npm run research:roi-analysis
```
